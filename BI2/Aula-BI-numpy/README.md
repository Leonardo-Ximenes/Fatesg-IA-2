Este é o repositório de atividades e exercícios práticos da disciplina de **Business Intelligence**, utilizando a biblioteca **NumPy** em Python, oferecida pela **Faculdade SENAI FATESG**.

## 📁 Estrutura do Arquivos

- `funcoes.py`: Arquivo base em Python destinado à implementação das funções e exercícios propostos ao longo das aulas.

## 🚀 Tecnologias e Bibliotecas

- [Python](https://www.python.org/)
- [NumPy](https://numpy.org/)

## 📖 Objetivo

O foco deste repositório é consolidar os conhecimentos aplicados em Business Intelligence por meio da manipulação, análise e processamento de dados numéricos, aproveitando a eficiência das estruturas de arrays e operações vetorizadas disponíveis na biblioteca NumPy.
